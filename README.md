# Project Setup

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Sitouxz/testcase-jcamp.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd testcase-jcamp
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Start the development server:**

   ```bash
   npm start
   ```
